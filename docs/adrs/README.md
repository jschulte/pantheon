# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) documenting key design decisions in Pantheon.

ADRs are immutable once accepted. To change a decision, create a new ADR that supersedes the old one.

## Template

```markdown
# ADR-NNN: Title

**Status:** Proposed | Accepted | Deprecated | Superseded by ADR-NNN
**Date:** YYYY-MM-DD

## Context
What is the issue that we're seeing that is motivating this decision?

## Decision
What is the change that we're proposing and/or doing?

## Consequences
What becomes easier or more difficult to do because of this change?
```
