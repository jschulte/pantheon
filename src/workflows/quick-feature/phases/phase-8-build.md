## Step 2: Execute Build Handoff

```
IF BUILD_MODE == "sequential":
  → Log: "Handing off to batch-stories in sequential mode..."
  → Load the batch-stories workflow:
    1. Read {project-root}/_bmad/core/tasks/workflow.xml
    2. Pass {project-root}/_bmad/pantheon/workflows/batch-stories/workflow.yaml as workflow-config
    3. Pre-select: mode = sequential, stories = VALID_STORIES (all valid story keys)
  → The batch-stories workflow takes over from here

ELSE IF BUILD_MODE == "parallel":
  → Log: "Handing off to batch-stories in parallel mode..."
  → Load the batch-stories workflow:
    1. Read {project-root}/_bmad/core/tasks/workflow.xml
    2. Pass {project-root}/_bmad/pantheon/workflows/batch-stories/workflow.yaml as workflow-config
    3. Pre-select: mode = parallel, stories = VALID_STORIES (all valid story keys)
  → The batch-stories workflow takes over from here

  → Display story file paths for manual review
  → Provide command to run when ready:
```
